# Inter Font Dosyaları

Bu klasörde Inter font dosyaları bulunmaktadır. Inter, açık kaynaklı bir font ve San Francisco'ya benzer bir görünüme sahiptir.

## Mevcut Font Dosyaları:

1. **Inter-Regular.ttf** - Normal metin için
2. **Inter-Medium.ttf** - Orta kalınlık metin için
3. **Inter-SemiBold.ttf** - Yarı kalın metin için
4. **Inter-Bold.ttf** - Kalın metin için

## Font Kullanımı:

Font dosyaları otomatik olarak yüklenir ve uygulama genelinde kullanılır. `fontWeight` özelliğine göre doğru font varyantı otomatik olarak seçilir:

- `fontWeight: 'normal'` veya belirtilmemiş → Inter-Regular
- `fontWeight: '500'` veya `'medium'` → Inter-Medium
- `fontWeight: '600'` veya `'semibold'` → Inter-SemiBold
- `fontWeight: 'bold'` veya `'700'` → Inter-Bold
